ZELANIX AG WEBSITE

INSTALLATION

1. Download all royalty-free images.
2. Save them into the images folder.
3. Replace logo.png with the final Zelanix Ag logo.
4. Open index.html in a web browser.

PUBLISH ONLINE

NETLIFY

1. Create a Netlify account.
2. Drag the entire Zelanix Ag folder onto Netlify Deploy.
3. Your website will be live instantly.

HOSTINGER

1. Purchase hosting.
2. Open File Manager.
3. Upload all website files to public_html.
